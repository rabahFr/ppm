Il a pour vocation, le développement d'une librairie pour faciliter l'usage du format image PPM.
Ceci dans le cadre d'un projet pédagogique, pour l'UE de programmation Rust, and 4ème année Big Data et IA.

La librairie permet de lire des fichiers PPM sous format P3 les sauvegarder sur disque et effectuer différentes opérations sur celles ci:
-Inverser l'image.
-Transformer l'image en nuances de gris

***MODE DEMPLOI***
-lancer le chargement et traitement d'image: "Cargo +nightly 'path/to/img'" 
-lancer le mode menu basique: "Cargo +nightly run"
-lancer test/bench: "Cargo +nightly "test OU bench"
La librairie sera portée sur Python, incessamment sous peu.
